# نور آية 🌟

**الشعار:** من كل آية.. نور وهداية

تطبيق تأملي يعرض آية من القرآن الكريم يوميًا مع تفسير موجز، تدبر، ودعاء، بالإضافة إلى أدوات لتسهيل التأمل والتفاعل مع الآيات.

---

## ✨ المميزات الرئيسية

### 🎯 المميزات الأساسية
- **آية اليوم** - عرض آية جديدة يومياً مع التدبر والدعاء
- **دعاء اليوم** 🆕 - 110 دعاء متنوع مع أولوية للمناسبات الخاصة
- **تصنيف ذكي** - 12 فئة (صبر، توكل، يقين، شكر، هداية، ثبات، رحمة، خوف، رجاء، دعاء، تدبر، عام)
- **المفضلة** - حفظ الآيات والأدعية مع إمكانية الحذف بالسحب 🆕
- **البحث المتقدم** - بحث في الآيات والأدعية مع تمييز النتائج 🆕
- **الوضع الليلي** - واجهة مريحة للعين مع دعم كامل للثيمات
- **مشاركة ونسخ** - مشاركة الآيات والأدعية كنص أو كصورة مصممة 🆕
- **تنقل سلس** - التمرير بين الآيات بسهولة (Swipe Navigation)
- **الورد اليومي** 🆕 - تحديد وتتبع ورد يومي مخصص من الآيات والأدعية

### 🌟 المميزات المتقدمة
- **دعم 5 لغات** 🌍
  - العربية 🇸🇦 | الإنجليزية 🇬🇧 | الفرنسية 🇫🇷 | الأوردو 🇵🇰 | الإندونيسية 🇮🇩
  - اختيار اللغة عند أول تشغيل
  - دعم تلقائي لـ RTL/LTR

- **8 خطوط قرآنية رسمية** ✍️
  - الرسم العثماني V22 (UthmanicHafs V22) - محدث 🆕
  - شهرزاد الجديد (Scheherazade New)
  - لطيف (Lateef)
  - أميري (Amiri) - الافتراضي
  - IBM Plex Arabic
  - Cairo
  - Noto Sans
  - Roboto

- **التقويم الهجري** 🌙
  - عرض وتبديل بين الهجري والميلادي
  - تلوين ذهبي خاص لشهر رمضان المبارك
  - أيقونة الهلال في رمضان

- **نظام الإنجازات الموسع** 🏆 🆕
  - 56 إنجاز متنوع (7 مستويات: برونزي، فضي، ذهبي، بلاتيني، ماسي، زمردي، أسطوري)
  - نظام نقاط تراكمية مع 7 رتب (مبتدئ → شيخ)
  - تتبع تلقائي للتقدم
  - عداد الأيام المتتالية (Streak Counter) 🔥
  - إنجازات للقراءة، المفضلة، المشاركة، الأدعية، الورد، والاستماع
  - نظام مصمم ليستمر أكثر من عام

- **التلاوة الصوتية** 🎧 🆕
  - 4 قراء مشهورين (المنشاوي، الحصري، العفاسي، عبد الباسط)
  - مشغل صوت مدمج مع تحكم كامل وشريط تقدم 🆕
  - روابط صوتية عالية الجودة من everyayah.com
  - تحكم مباشر من كارد الآية 🆕

- **الإشعارات اليومية** 🔔
  - استلم آية يومية في الوقت المحدد
  - آية عشوائية مع كل إشعار
  - إعدادات قابلة للتخصيص

- **تحديث المحتوى عبر الإنترنت** 🔄 🆕
  - مزامنة تلقائية أو يدوية للمحتوى الجديد
  - إحصائيات التحديثات (آيات جديدة، أدعية جديدة)
  - جاهز للربط مع Laravel API مستقبلاً

- **مشاركة كصورة مصممة** 🎨
  - **للآيات**: 5 قوالب تصميم احترافية
  - **للأدعية**: 4 قوالب تصميم احترافية 🆕
    - **كلاسيكي** 📜 - أنيق مع إطار وزخارف
    - **عصري** 🎨 - gradients وألوان متدرجة
    - **بسيط** ✨ - minimalist راقي
    - **إسلامي** 🕌 - أصيل مع زخارف إسلامية
  - دقة عالية (1080x1920 @ 2x) - محسن 🆕
  - معاينة مباشرة قبل المشاركة

## 📦 التثبيت والتشغيل

### المتطلبات
- Flutter SDK (3.24 أو أحدث)
- Dart SDK
- محرر كود (VS Code أو Android Studio)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone <repository-url>
cd noor_ayah
```

2. **تثبيت المكتبات**
```bash
flutter pub get
```

3. **تشغيل التطبيق**
```bash
flutter run
```

### إضافة الخطوط (اختياري لكن موصى به)

لتحسين التجربة البصرية، قم بتحميل الخطوط من:
- الروابط موجودة في `assets/fonts/README.md`
- ضع ملفات `.ttf` في مجلد `assets/fonts/`

## 🎯 الشاشات (20 شاشة)

### الشاشات الرئيسية
1. **MainScreen** - Container مع Bottom Navigation
2. **HomeScreen** - آية اليوم + دعاء اليوم + الورد اليومي + التقويم الهجري + الإنجازات + أيقونة التطبيق 🆕
3. **CategoriesScreen** - تصفح الآيات حسب الفئات (12 فئة)
4. **FavoritesScreen** - الآيات والأدعية المحفوظة مع تبويبات منفصلة 🆕
5. **SearchScreen** - بحث متقدم في الآيات والأدعية مع تمييز النتائج 🆕
6. **SettingsScreen** - تخصيص كامل للتطبيق مع أيقونة التطبيق 🆕
7. **VerseDetailsScreen** - عرض كامل مع التنقل بين الآيات ومشغل صوتي مدمج 🆕

### شاشات دعاء اليوم 🆕
8. **DuaOfDayScreen** - دعاء اليوم الذكي حسب الوقت والمناسبات
9. **DuaDetailsScreen** - تفاصيل الدعاء مع الفوائد والمصدر
10. **DuasListScreen** - قائمة 110 دعاء مع 3 تبويبات وبحث
11. **DuaSharePreviewScreen** - مشاركة الدعاء كصورة (4 قوالب)

### شاشات الورد والمزامنة 🆕
12. **DailyWirdScreen** - الورد اليومي مع تتبع الإنجاز
13. **SyncScreen** - مزامنة المحتوى مع إحصائيات التحديثات

### الشاشات الإضافية
14. **CategoryVersesScreen** - عرض جميع آيات فئة معينة
15. **LanguageSelectionScreen** - اختيار اللغة عند أول تشغيل
16. **LanguageSettingsScreen** - تبديل اللغة
17. **FontSelectionScreen** - اختيار خط الآيات مع معاينة
18. **NotificationSettingsScreen** - جدولة الإشعارات اليومية
19. **AchievementsScreen** - عرض 56 إنجاز مع النقاط والرتب
20. **SharePreviewScreen** - معاينة ومشاركة الآيات كصور
