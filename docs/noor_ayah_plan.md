# خطة تطوير تطبيق نور آية

**الشعار:** من كل آية.. نور وهداية

## نظرة عامة

تطبيق تأملي يعرض آية من القرآن الكريم يوميًا مع تفسير موجز، تدبر، ودعاء، بالإضافة إلى أدوات لتسهيل التأمل والتفاعل مع الآيات.

---

## 1. أهداف التطبيق

1. تقديم تجربة يومية للمستخدم للاطلاع على آية من القرآن والتدبر فيها
2. تسهيل حفظ ومشاركة الآيات مع الآخرين
3. توفير واجهة هادئة ومريحة للقراءة والتأمل
4. دعم المستخدمين في بناء عادة يومية للتدبر والتأمل
5. جمع محتوى ديني موثوق بطريقة منظمة وسهلة الوصول

---

## 2. الجمهور المستهدف

- المسلمين من جميع الأعمار، خاصة الشباب
- مستخدمي الهواتف الذكية الباحثين عن محتوى ديني يومي مختصر
- المهتمين بالتأمل الروحي والهداية اليومية

---

## 3. المميزات الأساسية (MVP)

| الميزة | الوصف |
|--------|-------|
| **آية اليوم** | تظهر آية جديدة يوميًا مع التدبر والدعاء |
| **عرض التدبر** | عند الضغط على الآية يظهر النص الكامل، التدبر، والدعاء |
| **تصنيف الآيات** | حسب المواضيع: صبر، خوف، رزق، يقين، شكر، توكل |
| **المفضلة** | حفظ الآيات المفضلة للوصول السريع |
| **البحث** | البحث بالكلمة أو النص داخل الآية أو التدبر |
| **الوضع الليلي** | واجهة مريحة للعين للقراءة الليلية |
| **تنبيه يومي** | إشعار بآية اليوم مع إمكانية ضبط الوقت |
| **استماع** | تشغيل تلاوة الآية (اختياري) |
| **مشاركة** | مشاركة الآية والتدبر كصورة أو نص على الشبكات الاجتماعية |

---

## 4. المميزات المتقدمة (مستقبلية)

- **الوِرد اليومي المخصص:** تحديد وقت يومي للتأمل في آية مختارة
- **خلفيات تفاعلية:** اختيار خلفيات متدرجة وألوان هادئة حسب الوقت أو المناسبة
- **نظام إنجازات وجوائز:** مكافآت افتراضية لاستمرار المستخدم على التدبر اليومي
- **إضافة الصوتيات:** إمكانية الاستماع لتفسير موجز للآية
- **تحديث المحتوى عبر الإنترنت:** جلب آيات جديدة أو إضافات بدون تحديث التطبيق
- **نسخة offline كاملة:** إمكانية قراءة جميع الآيات بدون إنترنت بعد تحميل المحتوى

## 8. تصور واجهات المستخدم (UI/UX)

### المبادئ العامة
1. **بساطة ووضوح:** كل واجهة تحتوي على عنصر أساسي للتركيز (الآية)
2. **ألوان هادئة وروحانية:** درجات الأخضر، البيج، الذهبي، مع خلفية متدرجة
3. **خط عربي واضح:** استخدام خط Amiri أو Uthman TN للآيات
4. **أيقونات واضحة وبسيطة:** مشاركة، مفضلة، تشغيل صوت، البحث، الإعدادات
5. **التنقل السلس:** شريط سفلي (Bottom Navigation Bar) للوصول السريع

3. الوضع الليلي يعكس خلفية داكنة مع نص فاتح للحفاظ على العين أثناء الليل
4. **Typography:** نص الآية أكبر من التدبر والدعاء، لتوجيه التركيز
5. لاحقًا يمكن إضافة تأثيرات ناعمة للانتقال بين الآيات عند التمرير
