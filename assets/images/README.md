# الصور والأيقونات

هذا المجلد مخصص للصور والأيقونات المستخدمة في التطبيق.

## الصور المقترحة
- `app_logo.png` - شعار التطبيق (512x512 بكسل)
- خلفيات متدرجة (اختياري)
- أيقونات الفئات (اختياري - حالياً نستخدم emojis)

## ملاحظة
التطبيق يعمل حالياً بدون صور إضافية، حيث نستخدم:
- أيقونات Material Design
- رموز Emoji للفئات
- ألوان متدرجة (Gradients) للخلفيات

يمكنك إضافة صور مخصصة لاحقاً لتحسين التجربة البصرية.
