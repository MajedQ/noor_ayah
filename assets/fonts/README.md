## 🎨 استخدام الخطوط في التطبيق

### الخطوط القرآنية (للآيات)
يمكن للمستخدمين اختيار الخط المفضل لديهم من خلال:
```
الإعدادات > المظهر > خط الآيات
```

الخيارات المتاحة:
1. **الرسم العثماني** - للحصول على مظهر المصحف الشريف
2. **شهرزاد** - خط عربي جميل مع تشكيل ممتاز
3. **لطيف** - خط نسخي أنيق
4. **أميري** - الخط الافتراضي (موصى به)

### خطوط الواجهة (حسب اللغة)

| اللغة | الخط الافتراضي | البديل |
|------|----------------|---------|
| العربية (ar) | Cairo | Amiri |
| الأردية (ur) | Noto Nastaliq Urdu | Amiri |
| الإنجليزية (en) | Roboto | Noto Sans |
| الفرنسية (fr) | Roboto | Noto Sans |
| الإندونيسية (id) | Roboto | Noto Sans |

## 📱 التكامل التقني

### في `pubspec.yaml`
تم تعريف جميع الخطوط في ملف `pubspec.yaml` مع التوزيعات الصحيحة:
- خطوط عربية قرآنية (5 عائلات)
- خطوط الأردية (1 عائلة)
- خطوط لاتينية (2 عائلة)

### في `font_provider.dart`
تم إنشاء `FontProvider` ذكي يختار الخط المناسب تلقائياً حسب:
1. **اللغة الحالية** - يختار الخط الأمثل للغة المستخدم
2. **تفضيلات المستخدم** - يحفظ اختيار المستخدم للخطوط
3. **السياق** - يميز بين خطوط الآيات وخطوط الواجهة

### الدوال المتاحة
```dart
// الحصول على خط الآية القرآنية المختار
String fontFamily = fontProvider.fontFamily;

// الحصول على خط الواجهة حسب اللغة
String uiFontFamily = fontProvider.uiFontFamily;

// الحصول على خط مخصص للغة معينة
String localeFont = fontProvider.getLocalizedUIFont('ur');

// الحصول على خط للنصوص العربية
String arabicFont = fontProvider.getArabicTextFont('ar');
```
